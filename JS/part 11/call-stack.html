Call Stack

In JavaScript, the call stack is a mechanism for keeping track of function calls. It operates in a last-in, first-out (LIFO) manner, meaning that the most recently called function is the first one to be executed.

When a function is called, it is added to the top of the call stack. If that function calls another function, the new function is added to the top of the stack. When a function finishes executing, it is removed from the stack, and control returns to the previous function.
